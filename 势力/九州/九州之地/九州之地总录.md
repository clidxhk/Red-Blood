## 九州咬合

九州之地并不是九块各自关门过活的孤州。

- [[雍天州]]居中掌印，离不开[[河梁州]]输粮、[[朔垣州]]挡锋与[[蘅泽州]]替它续清议
- [[朔垣州]]与[[雪嶂州]]共同托住北与西北的险线，一个像刀，一个像关
- [[河梁州]]是九州血路，向北喂兵，向南喂城，也把[[绫水州]]与中枢缝在一起
- [[绫水州]]与[[沧门州]]一江一海，最容易把新术、新器与新政一并带进九州
- [[蘅泽州]]把许多硬事磨成软话，是南地最深的解释层
- [[炎岫州]]与[[榕雨州]]共掌山火、药脉与贴地暗网，是最容易长出新生力量的南方底骨
- [[沧门州]]开海，亦开祸；它带来变化，也带来更细的新依附

## 主要通路

- **北线兵粮路**：[[河梁州]] -> [[朔垣州]] -> 北境诸关
- **近畿中轴路**：[[河梁州]] -> [[雍天州]] -> 诸州官册与朝令
- **江海新术路**：[[沧门州]] -> [[绫水州]] -> [[雍天州]]与诸南州
- **南山火药路**：[[炎岫州]] -> [[蘅泽州]] / [[榕雨州]] -> 外运诸州
- **高关险道线**：[[雪嶂州]] <-> [[朔垣州]] <-> [[雍天州]]西北门

## 总体格局

若把九州看成一具会呼吸的旧身躯，那么：

- [[雍天州]]是心口
- [[河梁州]]是血脉
- [[朔垣州]]与[[雪嶂州]]是肩背与骨关
- [[绫水州]]与[[沧门州]]是耳目与外门
- [[蘅泽州]]是喉舌
- [[炎岫州]]与[[榕雨州]]则更像腹中地火与深处湿脉

也正因为九州各州咬得如此紧，任何一州出问题，都很难只是一州出问题。
