---
name: Crosslink Filename Exactness
description: 全仓库交叉引用必须严格匹配仓库文件名，不得擅自改字、补空格或换标点
type: feedback
---

规则：全仓库所有 `[[...]]` 交叉引用都必须严格照仓库实际文件名书写，不得擅自改字、补空格、换标点或调整编号格式；例如应写 `[[11.茶馆闲聊·火楼]]`，不可写成 `[[11. 茶馆闲聊·火楼]]`。

**原因：** 用户明确要求所有交叉引用名与真实文件名完全一致，避免 Agent 自作主张改格式，造成链接失配、风格混乱和仓库命名不统一。

**应用场景：** 新写、改写、回填任何含 `[[...]]` 交叉引用的正文、说明文或 Agent 规则时，适用于全仓库所有系列、人物、器物、场景与说明文件。
