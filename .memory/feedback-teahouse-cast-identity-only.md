description: 茶馆闲聊的出场人物栏目只写人物身份，不写视角功能或叙事职责
type: feedback
---

规则：以后写茶馆闲聊时，“出场人物”栏目只写人物名和身份，不写“负责什么视角”“承担什么功能”“本回说什么位置”之类说明。

规则：人物在这一回里的作用、话头分量和叙述职责，都放到正文自然体现，不在人物列表里提前解释。

**原因：** 用户明确要求，出场人物只需要身份，不需要视角说明；人物列表若写成功能表，会削弱正文自然感。

**应用场景：** 处理茶馆闲聊、多人同场街谈或同类固定结构章节时，在“出场人物”栏目仅保留姓名与身份短语，如“女医修”“掌柜”“账房”“短工”“卖簪钗的妇人”等。
