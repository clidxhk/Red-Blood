type: feedback

规则：撰写九州历史报告、王朝兴亡、起义考述时，优先使用“伪史考据体”而非纯概述体。正文宜带 `撰者记`、分节考述、残篇引文、州志口碑、军簿残册等史料化载体，让事件以互证、辨析、留白的方式展开，而不是直接站在作者幕后总结。

**原因：** 该项目现有高质量历史报告均通过旧谱、州志、残卷、民谣等材料感营造可信度；这种写法既保留古韵，又能承接大规模历史事件的复杂性。

**应用场景：** 新增或修订历史报告、皇朝末世、遍地大起义、帝阙制度沿革等条目时，优先采用考据体结构；尤其在涉及现实历史原型转译时，用“史料化叙述”替代直接照搬史实。
