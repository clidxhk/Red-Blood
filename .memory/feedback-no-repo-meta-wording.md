---
name: No Repo Meta Wording
description: 交付文本不得出现仓库外视角表述，如“仓库里已有文档”“仓库中写过”
type: feedback
---

规则：交付给用户的设定正文、概述、分析与补写说明，不得出现“仓库里已有文档”“仓库中写过”“仓库里早有设定”“当前仓库没有”这类站在仓库外评论文本状态的说法。

**原因：** 用户明确指出这类说法会让口吻立刻出戏，把文本从世界内部叙述和正常协作说明拉回生硬的仓库管理视角。

**应用场景：** 回答设定问题、扩写条目、补写说明、分析依据和交付修改结果时；需要交代依据时，改用更自然的执行表述，例如“现有设定暂无直接依据”或“前文已见其名”。
