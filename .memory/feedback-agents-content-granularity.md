---
name: AGENTS Content Granularity
description: 内容应按单一概念拆分写入 AGENTS 约束内，单篇控制在 300 行以内，避免多概念混写
type: feedback
---

规则：写入 AGENTS 约束下的内容时，单篇尽量只承载一个概念；能拆开的设定不要并在同一篇里写。每篇内容控制在 300 行以内，优先通过拆分条目来保持结构清晰。

**原因：** 用户明确要求“把内容写作AGENTS里，一篇内容不能超过300行，然后尽量不要把两个概念写在一个里面。能拆开就拆开清晰描述”。这样更便于检索、回填和后续扩写。

**应用场景：** 适用于新增或改写世界观、势力、器物、人物、历史、地域等条目时的文件拆分与篇幅控制；若内容接近上限或出现两个独立概念，应优先拆成关联文件并互相链接。
