规则：后续撰写“茶馆闲聊”系列时，章节结构要比普通场景文更稳定、清楚，优先采用“背景概述 / 核心设定 / 出场人物 / 茶馆情状 / 席间杂论 / 余意”这一类职责明确的段名；允许有文气，但不要为求花样牺牲体系感。

规则：茶馆条目开头先交代本回夜话因何而起，再用“核心设定”单独钉清这一回真正要掰开的秩序问题、修行规则或阶层处境，然后再说明由谁来承接这场议论；环境段需专门写出今夜茶馆与平日不同的气压、声气与门外风声，不能把人物介绍、背景说明和馆内情状混成一团。

规则：正文主段应把不同阶层与位置的人物视角分层推进，结尾则必须把这场夜话收束成更大的世界观问题或后续线索，不可只停在“众人聊了一场”。

**原因：** 用户先前要求把茶馆闲聊整理成清晰稳定的系列结构，本次又明确要求在“背景概述”之下增设“核心设定”，让每回夜话先钉清真正要展开的世界观骨架。

**应用场景：** 处理“场景/茶馆闲聊”目录下的新写、重写与格式整理任务时，优先按稳定段落职责组织全文；若需改名，可在不改变职责的前提下微调栏目名，但避免每篇各用一套段名。
