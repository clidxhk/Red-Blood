---
name: State And Reference Split
description: 州域等双文件设定采用正文官面总录、参考稿民间流传与异闻补注的分工
type: feedback
---

规则：成对出现的设定正文与参考稿应主动分工。正文优先采用较正式、官修、州志或总录口吻，负责稳定骨架、地理、人情、势力与制度；参考稿则优先采用民间流传、异闻补注、海上传闻、口碑旧说等角度，补足奇幻感与边角信息，避免与正文大段重复。

**原因：** 用户明确提出州域条目这样拆分会更好，认为正式版本与民间版本并存，既能保证清晰度，也能增强世界的层次和神秘感。

**应用场景：** 今后处理州域、势力、地域志、总录类双文件结构时，若同时存在 `正文` 与 `参考`，默认按“官方骨架 + 民间补充”双轨组织内容。
