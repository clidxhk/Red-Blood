---
name: Character Entry Logic
description: 茶馆等群像场景里，既有角色的出场要顺着其既有职责、边界与行动链进入，避免偶遇式硬接
type: feedback
---

规则：处理既有角色，尤其是边界感强、职责明确的人物时，必须让其通过既有事务、职业动作、关系链或空间位置自然进入场景，不能只因“刚好路过”“被人招手叫进来”就承担关键戏份。

**原因：** 用户指出柳青蘅加入火楼一篇的逻辑不对，说明角色入场若脱离既有人设边界，会削弱人物可信度与场景因果。

**应用场景：** 茶馆闲聊、群像夜话、人物插入既有场景、重写既有剧情段落时，尤其适用于柳青蘅、老舒、邢五这类功能位置稳定的人物。
